# PasswordManager
Project for Android cource on Kotlin

My application provides the following functionality:
* The ability to add/edit/remove passwords;
* The ability to log in to the application using Firebase;
* The ability to change password for app account;
* The ability to generate random password;
* The ability to filter password list.
